During my time in **Workforce Bootcamp - First Cloud AI Journey**, I had the opportunity to learn, practice, and apply AWS knowledge to the team project **AI Job Matching Platform**. The project uses an AWS Serverless architecture and combines services such as Amazon Cognito, API Gateway, Lambda, DynamoDB, S3, and AI services to support job search and CV evaluation workflows.

In the project, my main contributions focused on **Cognito Authentication**, **API Gateway with JWT Authorizer**, **frontend authentication integration**, and participation in the **CV matching flow**. I learned how Cognito authenticates users, how JWT Authorizer protects API routes, how the frontend manages authentication state, and how the CV Matching Flow connects frontend actions, API Gateway, Lambda, S3, DynamoDB, and AI-based CV evaluation steps. For broader flows such as CV Matching, I contributed to and helped refine parts of the workflow rather than claiming full ownership of the entire system.

Through this process, I improved my understanding of AWS Serverless, API design, user authentication, data handling, and technical documentation. I also strengthened my ability to understand requirements, analyze issues, test end-to-end workflows, and present workshop content in a clearer and more structured way.

Regarding work attitude, I tried to maintain responsibility, proactiveness, and collaboration with team members. When facing an issue or an unclear requirement, I usually checked it first, then discussed it with the team to align on the solution. I still need to improve my technical communication and root-cause analysis skills, but overall I maintained a serious attitude and a growth mindset throughout the project.
