Trong phần này, chúng ta sẽ tiến hành đăng ký và lấy các mã khóa API (API Key) từ các dịch vụ bên thứ ba để tích hợp tính năng tìm kiếm và trí tuệ nhân tạo (AI) vào dự án.

---

### 1. Lấy API Key từ SerpAPI

SerpAPI được sử dụng để truy vấn kết quả tìm kiếm của các công cụ tìm kiếm một cách lập trình hóa.

#### 1.1. Đăng nhập hoặc Đăng ký:
Truy cập trang chủ [SerpAPI](https://serpapi.com/) để đăng nhập hoặc đăng ký tài khoản mới.

![SerpAPI Login Page](/images/5-Workshop/5.2-Prerequisite/5.2.3-Third-party-APIs/serpapi.png)

#### 1.2. Lấy API Key:
Điều hướng đến trang quản trị tài khoản (dashboard) hoặc trang API key và sao chép khóa API cá nhân của bạn.

![SerpAPI API Key](/images/5-Workshop/5.2-Prerequisite/5.2.3-Third-party-APIs/serpapi-key.png)

---

### 2. Lấy Gemini API Key

Gemini API được sử dụng để tích hợp các tính năng AI tạo sinh (Generative AI) vào dự án.

#### 2.1. Truy cập Google AI Studio:
Truy cập trang quản lý mã khóa tại [Google AI Studio API Keys](https://aistudio.google.com/app/apikey).

![Google AI Studio API Keys](/images/5-Workshop/5.2-Prerequisite/5.2.3-Third-party-APIs/google-ai-studio.png)

#### 2.2. Tạo và Sao chép API Key:
Đăng nhập bằng tài khoản Google của bạn, nhấn vào nút **Create API key** (Tạo khóa API), chọn hoặc tạo một dự án liên kết, sau đó sao chép mã khóa API vừa được tạo.

![Create Gemini API Key](/images/5-Workshop/5.2-Prerequisite/5.2.3-Third-party-APIs/gemini-api-key.png)
